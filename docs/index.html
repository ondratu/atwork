AtWork

A simple time counter dialog for tracking your work hours

Python 3.9+ tkinter Cross-platform

üìñ About

AtWork is a lightweight desktop application designed to help you track your working hours efficiently. It provides a simple graphical interface where you can input your start time, pause duration, and required working hours, and it automatically calculates when you can go home.

Key Features:

  • Real-time calculations: Automatically updates the current time and calculates worked hours
  • Flexible inputs: Customize start time, pause duration, and working hours
  • Visual feedback: Color-coded display shows if you've completed your working hours
  • Simple interface: Clean and intuitive tkinter-based GUI
  • Lightweight: Minimal resource usage, perfect for running in the background

üöÄ Installation

Prerequisites

Before installing AtWork, ensure you have the following dependencies:

  • Python 3.9 or higher - The application requires Python 3.9+
  • tkinter - Python's standard GUI library (usually included with Python)

Installing Python and tkinter

On Ubuntu/Debian:

sudo apt update
sudo apt install python3 python3-tk

On Fedora/RHEL:

sudo dnf install python3 python3-tkinter

On macOS:

# Using Homebrew
brew install python-tk@3.9

# Or if Python is already installed:
# tkinter is usually included with Python on macOS

On Windows:

Download Python from python.org. Make sure to check "Add Python to PATH" during installation. tkinter is included by default.

Installing AtWork

Option 1: Install from source (recommended for development)

# Clone the repository
git clone https://github.com/ondratu/atwork.git
cd atwork

# Install in development mode
pip install -e .

Option 2: Install directly with pip

# Install from PyPI (if published)
pip install atwork

# Or install directly from GitHub
pip install git+https://github.com/ondratu/atwork.git

Verifying Installation

After installation, you can verify that AtWork is installed correctly:

atwork --version  # If this doesn't work, try running directly:
python -m atwork

üí° Usage

Running AtWork

Once installed, you can launch AtWork in several ways:

Command Line:

# Using the installed script
atwork

# Or using Python module
python -m atwork

Desktop Launcher (Linux):

On Linux systems, AtWork installs a desktop entry file. You can find it in your application menu under the "Office" category as "At Work".

Using the Application

  1. Start time: Enter the time you started working (default: 7:30)
  2. Actual time: Shows the current time (updates automatically)
  3. Pause time: Enter your total break/lunch duration (default: 0:30)
  4. Working time: Enter your required working hours (default: 8:00)
  5. Press Enter: After entering values, press Enter to recalculate

The application will display:

  • Worked time: How long you've been working (excluding breaks)
  • Time to go: How much time until you can leave (red) or overtime (blue)
  • When to go: The exact time you can leave work
